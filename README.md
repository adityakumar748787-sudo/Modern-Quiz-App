🧠 QuizMaster – Modern Quiz Web App

A fully responsive, modern quiz application built using HTML, CSS, and JavaScript.

It includes a timer, progress bar, dark mode, local storage high score system, and smooth UI with glassmorphism design.

🚀 Features

⏳ Per-question countdown timer

📊 Dynamic progress bar

🏆 High score saved using LocalStorage

🌙 Dark / Light mode toggle

🔀 Questions shuffle every time

📱 Fully responsive design

🎨 Modern glassmorphism UI

🔁 Restart functionality

🛠 Tech Stack

HTML5

CSS3 (Flexbox + Modern UI Styling)

Vanilla JavaScript (ES6)

LocalStorage API
